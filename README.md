# Sales Performance Analysis Dashboard (Power BI)

## Overview
This project analyzes sales data to generate business insights and support decision-making using Power BI.

## Dataset
- 10,000+ sales records
- Includes revenue, shipments, products, and regions

## Tools Used
- Power BI
- DAX
- Power Query
- SQL
- Excel

## Key Metrics
- Revenue: $14M
- Profit: $1M
- Profit %: 57%

## Dashboard Features
- KPI Cards for Revenue, Profit, and Shipments
- Product-wise and region-wise analysis
- Salesperson performance tracking
- Interactive filters and slicers

## Key Insights
- Identified top-performing products contributing over 40% of revenue
- Analyzed shipment distribution trends
- Derived business insights for performance improvement
